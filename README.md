# Python Streamlit 记账工具

这是一个使用 Python + Streamlit + sqlite3 的简单网页记账工具。

## 功能
- 添加账目：金额、分类、日期、备注
- 查看账目列表：支持按月份和分类筛选
- 删除账目：输入账目 ID 删除
- 分类统计：显示柱状图和统计表

## 运行步骤
1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
2. 启动应用：
   ```bash
   streamlit run app.py
   ```

## 项目结构
- app.py：主界面和页面布局
- db.py：数据库初始化与 CRUD 操作
- services.py：业务逻辑和格式化
- models.py：分类常量
- finance.db：SQLite 数据库文件
